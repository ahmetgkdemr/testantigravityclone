# YOGII

Premium comfort wear brand website built with modern web technologies.

[![Next.js](https://img.shields.io/badge/Next.js-14-black?logo=next.js)](https://nextjs.org/)
[![TypeScript](https://img.shields.io/badge/TypeScript-5-blue?logo=typescript)](https://www.typescriptlang.org/)
[![Tailwind CSS](https://img.shields.io/badge/Tailwind_CSS-3-38bdf8?logo=tailwindcss)](https://tailwindcss.com/)

## Features

- **Home** — Hero section with particle effects, product features, and CTA
- **Collections** — Browse product categories and new arrivals
- **Pricing** — Tier comparison, feature table, and FAQ
- **Blog** — Articles with filtering, search, and dynamic post pages
- **About** — Brand story, stats, team, values, and timeline
- **Contact** — Contact form, support channels, and FAQ
- **Docs** — Size guide, care instructions, materials, and styling tips

## Tech Stack

- **Next.js 14** (App Router) + **TypeScript**
- **Tailwind CSS** for styling
- **Framer Motion** for animations
- **Lucide React** for icons

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
